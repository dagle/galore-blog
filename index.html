Often when we work with email, it’s handy to not just verify email or decrypt the email but to add, manage and delete keys. Galore doesn’t come with such tools in of it self but it’s very easy to add one. Add gpg-tui gpg-tui is a cool interface to gpg. So we add that to neovim. To add it, start by installing gpg-tui and then we install toggleterm to run gpg-tui inside of it.
